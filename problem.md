# Round 1 – Coding Challenge

## Problem: Sum of Two Numbers

### Description
Given two integers `a` and `b`, compute and print their sum.

### Input Format
Two space-separated integers.

### Output Format
Print a single integer – the sum of the two numbers.

### Constraints
- -10^6 ≤ a, b ≤ 10^6
- Time limit per test case: 2 seconds

### Example
Input:
2 3

Output:
5
